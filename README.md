# Go-FlowerViewing
꽃놀이 가자!!

* 드디어 봄 노래 전용 무료 스트리밍 서비스 '꽃놀이 가자' 안드로이드 버전이 출시되었습니다!!
* 앱 다운로드 : http://kaywa.me/t5PBF

* '노래로 봄 분위기 좀 내보고 싶은데 노래 한곡 한곡 다운받아 리스트 만들기는 귀찮고... 누가 진짜 좋은 노래만 엄선해서 대신 만들어주면 좋겠는데...'
* 꽃놀이 가자는 '절대음감 아티스트'가 봄 분위기에 딱 맞은 음악을 한곡 한곡 선정하여 모아놓은 봄노래 모음 어플입니다.
* 저희 서비스는 유튜브 동영상 API를 이용해 만든 무료 스트리밍 서비스입니다.
* 단지 터치 한 번으로 당신이 과거 어디선가 들었던, 꼭 한 번 다시 듣고 싶었던 이름 모를 아름다운 노래를 다시 듣게 될 것임을 약속합니다.
